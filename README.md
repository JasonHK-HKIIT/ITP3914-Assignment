# ITP3914 Assignment

This project is a Java-based Gomoku game developed for the ITP3914 Programming module assignment. It runs in console mode on a 10&times;10 board, uses `1` and `2` to represent the two players, and supports turn-based play with input validation, board redrawing, and win/draw detection for four consecutive stones in a row, column, or diagonal.
